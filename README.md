# 📌 Projeto Final - Algoritmos em Python

Sistema de menu interativo desenvolvido em Python.

## 🔧 Funcionalidades

- Cadastro de dados
- Consulta de registros
- Manipulação de arquivos
- Interface modularizada
- Estrutura organizada em múltiplos arquivos

## 🛠 Tecnologias utilizadas

- Python 3
- Estrutura modular
- Manipulação de arquivos

## 📂 Estrutura do Projeto

- main.py → Arquivo principal
- interface.py → Interface/menu
- cadastro.py → Cadastro de dados
- consulta.py → Consulta de dados
- arquivo.py → Manipulação de arquivos
- dicao.py → Estrutura de dados
